# easyteams

This is a FlexOS project. It contains a prototype, design system, and documentation for easyteams.

## Getting Started

To get started with this project, follow the instructions in the `README.md` file in the root directory.

## Project Structure

- `prototype/`: Contains the interactive prototype of the application.
- `design/`: Contains the design system documentation and assets.
- `docs/`: Contains additional project documentation.

## Commands

- `flexos dev`: Starts the development server for the prototype.
- `flexos build`: Builds the prototype for production.
- `flexos lint`: Lints the project files.
- `flexos test`: Runs tests.

For more information on FlexOS, visit [flexos.dev](https://flexos.dev).
