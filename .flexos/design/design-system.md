# Design SystemThis document outlines the core design principles and components for **easyteams** to ensure consistency and a cohesive user experience.## Color PaletteThe color palette is derived from the primary brand color, `#ec4899`, and includes a range of shades for various UI elements, as well as an accent color.### Light Mode- **Primary**: `#ec4899` (Used for main interactive elements, branding)- **Primary Dark**: `#d42b82` (Used for hover states, active states of primary elements)- **Primary Light**: `#f8b3d6` (Used for backgrounds, subtle highlights)- **Accent**: `#48ec99` (Used for secondary interactive elements, highlights, success states)### Dark Mode- **Primary**: `#f6b3d6` (Used for main interactive elements, branding)- **Primary Dark**: `#ec4899` (Used for hover states, active states of primary elements)- **Primary Light**: `#fce4f0` (Used for backgrounds, subtle highlights)- **Accent**: `#b3f6d6` (Used for secondary interactive elements, highlights, success states)### Color
